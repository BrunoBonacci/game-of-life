# game-of-life

Code from a Clojure Dojo session to build the Conway's Game Of Life.

![pulsar](pulsar.gif)

## Usage

Data files are available under `./data` directory.
You can change the speed by setting the environment variable `RATE`.

```
export RATE=5
lein run [data-file.edn]

```
